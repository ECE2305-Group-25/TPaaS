## Running

The server requries that the following components be installed on your computer:

* Python 3.7+
* Flask 0.12.5+

To launch the server:

**Linux/Mac:**
```bash
$ cd <project_root>/TPaaS/server
$ ./server.sh
```

**Windows:**
```cmd
> explorer "https://wiki.archlinux.org/index.php/Installation_guide"
```
